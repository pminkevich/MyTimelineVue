### My Timeline Vue

- Easy to add  in your project.
- Easy to use and customize.


#How to add?
-Add timeline.css file
```html
 <link rel="stylesheet" href="css/mycss.css">
```
-Add timeline.js file
```html
 <script src="js/timeline.js"></script>
```
-Add in your HTML file
```html
<timeline></timeline>
```
